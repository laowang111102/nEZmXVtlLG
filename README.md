## 前言

欢迎来到大学生心理健康测评管理系统小程序项目。本项目致力于为大学生提供心理健康测评服务，帮助他们更好地了解自己的心理状况，以便及时调整和改善。以下是对本项目的详细介绍。

## 内容介绍

本项目采用Java语言，结合Spring Boot、Spring MVC、MyBatis等主流框架，以及微信小程序、Uniapp等前端技术进行开发。系统主要包括用户管理、测评管理、报告管理等功能模块，为大学生提供全面的心理健康测评服务。

## 技术介绍

- **语言**：Java
- **使用框架**：Spring、Spring MVC、MyBatis、微信小程序
- **前端技术**：JS、Vue、CSS3、Uniapp
- **开发工具**：IDEA/Eclipse、Uniapp
- **数据库**：MySQL 5.7/8.0
- **数据库管理工具**：phpstudy/Navicat
- **JDK版本**：jdk1.8
- **Maven**：apache-maven 3.8.1-bin
- **前端环境**：Node.Js 12\14\16
